# T&TG Trade Corp — Django Web Application

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py makemigrations
python manage.py migrate

# Create admin superuser
python manage.py createsuperuser

# Run the development server
python manage.py runserver
```

## Admin Access
- URL: http://127.0.0.1:8000/admin/

## Features
- Global Marketplace (Local & International)
- Avon Points Loyalty System (5.5% consumer / 8.5% referral)
- Insurance, Brokerage & Forex Services
- Training Programs with Certificate system
- TV Program platform
- KYC document verification
- Multi-country support (CA, UG, NL, US, KE, JP)
- User Dashboard & Public Profiles
- Express & Ordinary delivery options
- Partner Company registration
